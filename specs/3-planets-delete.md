# Planets API

The Planets API is a simple REST api for testing against.

## `DELETE /api/dot/planets/{id}`

* API_KEY: "1234"

Deletes a new planet in the system.

===

### Example response

* Status: `200`
* Content-Type: `application/json`

## `GET /api/dot/planets/{id}`

* API_KEY: "1234"

Gets a planet in the system.

===

### Example response

* Status: `404`
* Content-Type: `application/json`
